# ani-cli-rus

Консольная утилита для поиска, просмотра и скачивания аниме на русском языке.

## Возможности

- Поиск аниме по названию через парсинг AnimeGO
- Онлайн просмотр выбранных серий в плеере (mpv/vlc)
- Скачивание серий для оффлайн-просмотра
- Библиотека скачанного — удобный доступ к загруженным файлам
- Хранение файлов в /tmp/ani-cli-rus

## Требования

Для работы программы необходимо установить:

1. Rust (компилятор и менеджер пакетов Cargo)
   - Установка: https://rustup.rs/
   - Проверка: rustc --version

2. Видеоплеер (один из следующих):
   - mpv (рекомендуется)
   - vlc
   - Установка (пример для Ubuntu/Debian):
     sudo apt install mpv
     # или
     sudo apt install vlc

## Установка и сборка

1. Клонируйте репозиторий:
   git clone https://github.com/Quarkubit/ani-cli-rus.git
   cd ani-cli-rus

2. Соберите проект:
   cargo build --release

3. Запустите:
   ./target/release/ani-cli-rus
   
   Опционально: Добавьте бинарный файл в PATH для глобального доступа:
   cp ./target/release/ani-cli-rus /usr/local/bin/
   ani-cli-rus

## Использование

При запуске вы увидите главное меню:

=== AniCli-Rus (AnimeGO) ===
1. Поиск аниме
2. Скачанное
q. Выход

### 1. Поиск аниме
- Введите название аниме для поиска.
- Выберите интересующий тайтл из списка.
- Выберите номер серии для просмотра.
- Вам будет предложено:
  - Смотреть онлайн (откроется в плеере)
  - Скачать (сохранится в /tmp/ani-cli-rus)

### 2. Скачанное
- Отображает список всех загруженных ранее серий.
- Позволяет воспроизвести файл без подключения к интернету.
- Файлы хранятся в формате: Название_аниме_Серия.mp4

### Структура хранения
Все скачанные файлы сохраняются в директорию:
/tmp/ani-cli-rus/
Убедитесь, что на диске достаточно свободного места для хранения видео.

## Устранение неполадок

Ошибка компиляции no method named 'flush':
- Убедитесь, что вы используете актуальную версию кода.
- Проблема решена добавлением use std::io::Write; в исходный код.

Файлы не скачиваются:
- Проверьте наличие свободного места в /tmp.
- Убедитесь, что у пользователя есть права на запись в директорию.

Видео не воспроизводится:
- Проверьте установлен ли плеер (mpv или vlc).
- Попробуйте запустить плеер отдельно с прямым путем к файлу.
